Playlist Context Menu

The majority of Black Omega’s functionality can be quickly accessed through the Context Menu. To view the Context Menu click the right mouse button inside the playlist view.

The Context Menu allows you to

  1. Add files to the playlist.
  2. Add a folder’s music files to the playlist.
  3. Access your iTunes Collection.
  4. Save the current playlist.
  5. Control playback.
  6. Cut, Copy and Paste audio tracks.
  7. Select All audio tracks.
  8. Remove selected tracks.
  9. Open the Preferences Dialog.
  10. Open the License Dialog.
