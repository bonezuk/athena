Copy, Cut and Paste

You can use the clipboard to move selected tracks around the playlist.

To copy tracks to the clipboard

  1. Select Edit > Copy
  2. Press Command (⌘) and C keys.
  3. From the Context Menu

To cut tracks into the clipboard

  1. Select Edit > Cut
  2. Press Command (⌘) and X keys.
  3. From the Context Menu

Tracks are pasted into the playlist either

  1. After the last selected track.
  2. Or at the end of the playlist if no track is selected.

To select Paste

  1. Select Edit > Paste
  2. Press Command (⌘) and V keys.
  3. From the Context Menu
