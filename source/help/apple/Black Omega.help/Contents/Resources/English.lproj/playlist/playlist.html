Playlist

The playlist shows the sequence of tracks to be played and allows you to edit it.

Tracks are displayed either grouped together by album or singly.

Black Omega uses embedded track information in order to display information about the track such as its name, artist, album and associated cover.

The playlist allows you to

  1. Add tracks.
  2. Edit the sequence of tracks.
  3. Start playing any track.
  4. Save playlist.

The track currently being played is highlighted in green with a speaker icon. A pause icon is shown when the track is paused.

The Context Menu can be accessed by clicking on the right mouse button
